# Fork ディレクトリ

このディレクトリには、元プロジェクト（KonomiTV）のフォークとして追加された、独自の機能とカスタマイズが含まれています。

## 概要

**Fork ディレクトリ**は、元プロジェクトのファイルを変更せずに、新機能を追加するための専用領域です。すべてのフォーク固有のコードは `Fork` プレフィックスを使用して命名されており、元プロジェクトとの明確な分離を実現しています。

## フォークの設計原則

### 1. 既存ファイル変更禁止
- 元プロジェクトのファイルは原則変更しない
- やむを得ず変更する場合は、コメントで変更箇所を追跡可能にする

### 2. 新規ファイルで実装
- すべての機能は `src/fork/` 配下に新規ファイルとして実装
- `Fork` プレフィックスを使用した統一命名規約を適用

### 3. 上流追従優先
- 元プロジェクトのアップデートに対応しやすい疎結合な構造
- 元プロジェクトとの結合点を最小限に抑える

## ディレクトリ構造

```
src/fork/
├── components/         # Fork固有のVueコンポーネント
├── composables/        # Fork固有のVue Composition API用ロジック
├── constants/          # Fork固有の定数定義
├── services/           # Fork固有のAPIクライアントとビジネスロジック
├── types/              # Fork固有のTypeScript型定義
├── utils/              # Fork固有のユーティリティ関数
└── views/              # Fork固有のルーティング可能なビューコンポーネント
```

各ディレクトリの詳細は、それぞれの`index.md`を参照してください。

## 現在の実装機能

### 録画番組表（全録タイムテーブル）
- **場所:** `components/ForkRecordedTimetable*.vue`, `views/ForkRecordedTimetableView.vue`
- **説明:** 録画番組を新聞式番組表形式で表示・管理する機能
- **主な特徴:**
  - ジャンル別色分け表示（左ボーダーストライプ + 背景グラデーション）
  - 時間帯ナビゲーション（朝、昼、夕、深夜）
  - 日付タブによる日付切り替え
  - レスポンシブデザイン
  - 詳細なツールチップ表示
  - 番組クリックによる動画再生

### その他の機能
- **録画番組リスト:** `ForkRecordedProgramList.vue`, `ForkProgramsView.vue`
- **動画選択:** `ForkVideoChoice.vue`
- **レスポンシブセレクト:** `ForkResponsiveSelect.vue`

---

**重要:** このディレクトリのすべての機能は、元プロジェクト（KonomiTV）のフォークとして独立して実装されています。元プロジェクトへの影響を最小限に抑え、上流のアップデートに追従しやすい設計を心がけてください。
