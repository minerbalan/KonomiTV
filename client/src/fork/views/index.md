# Views

このディレクトリには、ルーティング可能なビューコンポーネントが含まれています。

## ファイル一覧

### `ForkRecordedTimetableView.vue`
録画番組表ビュー（ルートビューコンポーネント）

- **機能:**
  - 録画番組データのAPI取得
  - チャンネルデータのAPI取得と並び替え
  - 日付選択に応じたデータ再取得
  - ローディング状態管理
  - 番組クリック時の動画再生処理
  - `ForkRecordedTimetable`コンポーネントの統合

- **API連携:**
  - `GET /api/recorded/programs`: 録画番組一覧取得
  - `GET /api/recorded/all-channels`: チャンネル一覧取得

**設計:** データ取得とビジネスロジックを担当し、プレゼンテーションは子コンポーネントに委譲。

### `ForkProgramsView.vue`
録画番組リストビュー

- **機能:**
  - 録画番組リストの表示
  - 各種フィルタリング・ソート機能

**用途:** リスト形式で録画番組を表示・管理するビュー。
